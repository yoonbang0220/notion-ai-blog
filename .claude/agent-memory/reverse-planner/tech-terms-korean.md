역기획서 작성 시 괄호 병기에 사용하는 표준 표현입니다.

| 기술 용어 | 쉬운 말 병기 |
|----------|------------|
| Next.js | 웹사이트를 빠르게 만들어주는 개발 도구 |
| React | 화면 요소를 조각처럼 만들어 조합하는 방식 |
| TypeScript | 코드 실수를 미리 잡아주는 엄격한 프로그래밍 언어 |
| Tailwind CSS | 미리 만들어진 디자인 조각을 조합하는 스타일 도구 |
| shadcn/ui | 미리 디자인된 화면 구성 요소 모음 |
| @base-ui/react | 기능만 있고 스타일은 없는 UI 기반 라이브러리 |
| App Router | 폴더 구조로 페이지를 관리하는 Next.js의 최신 방식 |
| Turbopack | Next.js의 매우 빠른 개발용 빌드 도구 |
| "use cache" | 서버 데이터 결과를 저장해 재활용하는 Next.js 16 기능 |
| next-themes | 다크모드 상태를 관리하는 라이브러리 |
| sonner | 화면 위에 알림 메시지를 띄워주는 라이브러리 |
| CVA (class-variance-authority) | 컴포넌트의 변형(크기, 색상 등)을 체계적으로 관리하는 도구 |
| API | 외부 서비스와 데이터를 주고받는 연결 통로 |
| 컴포넌트 | 화면을 구성하는 독립적인 재사용 가능한 블록 |
| 서버 사이드 렌더링 | 서버에서 화면을 미리 그려서 사용자에게 전달하는 방식 |
| 캐싱 | 자주 쓰는 데이터를 저장해두어 다시 불러오는 속도를 높이는 방식 |
| oklch | 밝기·채도·색상을 수학적으로 표현하는 현대적 색상 표기 방식 |
| OAuth | 구글, 깃헙 등 기존 계정으로 로그인할 수 있게 해주는 표준 방식 |
| Toast | 화면 한쪽에 잠깐 나타났다 사라지는 알림 메시지 |
| 접근성 | 장애가 있는 사용자도 불편 없이 사용할 수 있도록 배려한 설계 |
| 환경 변수 | 데이터베이스 주소, 비밀키 등 실행 환경에 따라 달라지는 설정값 |
| 비동기 | 기다리지 않고 동시에 처리하는 방식 |
| 페이지네이션 | 긴 목록을 여러 페이지로 나눠서 보여주는 방식 |
