- **PRD SSOT**: `docs/NOTION_BLOG_PRD.md` (현재 버전 2026-05-16 작성). 우선순위·기능 명세·DataTable 모두 여기서 출발.
- **로드맵**: `docs/ROADMAP.md` (2026-05-17 초기 작성). 변경 이력 섹션 유지.
- **타겟**: AI 입문자(비개발자 포함). 페르소나 "김지원, 28세 마케터".

**Why**: 향후 작업 시 PRD/ROADMAP 두 문서가 단일 출처임을 잊지 않기 위함.

**How to apply**: 새 기능 요청 시 먼저 PRD에 반영되어 있는지 확인, 없으면 PRD 갱신 → 로드맵 갱신 순서. Future Work 항목은 [[roadmap-conventions]] 의 우선순위 점수 기준으로 정렬.
